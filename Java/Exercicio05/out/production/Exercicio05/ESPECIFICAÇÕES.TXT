
Desenvolva um formulário de cadastro em Swing. 
O formulário deve conter os campos:
Nome, Endereço, UF, Email, CPF, Sexo (M, F, O), Preferências (Filme, Música, Teatro, Outros).

O formulário também deve conter dois botões:
 
Cancelar 
(se clicado pergunta se o usuário quer sair da aplicação, se confirmado, sai da aplicação) 
Confirmar 
(se clicado exibe os dados do formulário na tela para o usuário num JoptionPane)
